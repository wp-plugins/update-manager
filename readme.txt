== Description ==

Update Manager was a plugin to enable automatic update checking of plugins via the [wp-plugins.net](http://www.wp-plugins.net). Unfortunately the WP-Plugins API has been withdrawn and the plugin no longer works.

On the upside, chatter on <a href="http://comox.textdrive.com/pipermail/wp-hackers/2007-May/012575.html" rel="nofollow">wp-hackers mailing list</a> suggests that a <a href="http://comox.textdrive.com/pipermail/wp-hackers/2007-May/subject.html" rel="nofollow">plugin update checker is being developed</a> as part of the <a href="http://codex.wordpress.org/GSoC2007" rel="nofollow">Google SoC</a>.

I'll keep an eye out and keep you updated.
